# Калькулятор на Streamlit

Это приложение калькулятора создано с использованием Streamlit на Python.

## Функциональность

- Ввод двух чисел
- Выбор операции (+, -, *, /)
- Выполнение вычисления
- Обработка ошибки деления на ноль
- Переключение языка интерфейса (русский/английский)

## Использование

1. Введите первое число
2. Выберите операцию
3. Введите второе число
4. Нажмите "Вычислить"
5. Используйте боковую панель для переключения языка интерфейса

Приложение доступно по адресу http://localhost:8501
