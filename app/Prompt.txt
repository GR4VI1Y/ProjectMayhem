Роль: Ты — опытный Python-разработчик, придерживающийся принципов чистой архитектуры.
Задача: Разработай исходный код для веб-приложения на Streamlit для анализа и визуализации данных по продажам интернет-магазинов из файлов dataset_1.xlsx и task.pynb. 

Архитектурные требования:
Модульная разработка: Раздели код на несколько логических модулей. Общий объем каждого файла не должен превышать 400 строк.
app.py: Основной файл для запуска приложения. Он должен импортировать функции из других модулей и определять структуру интерфейса.
data_loader.py: Модуль с функцией для загрузки и кэширования данных из CSV-файла.
plotting.py: Модуль, содержащий функции для создания различных графиков (например, с помощью Plotly Express).
analysis.py: Модуль для функций анализа данных (например, расчет статистики, выявление аномалий).
Синтаксис и стиль: Весь код должен соответствовать стандарту PEP 8. Используй аннотации типов (type hints) для всех функций.Функциональные требования к приложению:
Заголовок: Приложение должно иметь заголовок "Анализатор Трафика Веб-сайта".
Загрузка данных: Приложение должно загружать данные из synthetic_traffic.csv. Используй кэширование (@st.cache_data) для предотвращения повторной загрузки.
Интерактивные элементы (сайдбар):
Выбор диапазона дат: Добавь st.date_input для выбора начальной и конечной даты анализа.
Выбор языка: русский, английский, китайский

Должна присутствовать возможность ввода своего массива данных на сайт для анализа с автоматическим пересчетом статистики, выводов и рекомендаций под него.
Также помимо основной страницы, должна быть страница с ответами на часто задаваемые вопросы (F.A.Q.), где будет рассказано о сайте и данных, которые он может принять для анализа.
Основная панель:
KPI-метрики: Выведи ключевые показатели (st.metric) за выбранный период: общее количество/сумма покупок, среднее количество/сумма покупок в день, максимальное количество/сумма покупок. Все должно быть адаптивно, чтобы можно было отфильтровать конкретные даты/города.
Интерактивный график: Отобрази линейный график сессий по дням с помощью plotly.express. График должен быть интерактивным (масштабирование, панорамирование).
Таблица с данными: Под графиком выведи DataFrame pandas с данными за выбранный период.Формат вывода: Предоставь код для каждого модуля (app.py, data_loader.py, plotting.py, analysis.py) в отдельных, четко обозначенных блоках кода.
