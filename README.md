# ProjectMayhem - Веб-приложение Streamlit для анализа продаж

Веб-приложение Streamlit для анализа и визуализации данных о продажах интернет-магазинов. Приложение поддерживает мультиязычность (русский, английский, китайский) и позволяет пользователям загружать собственные данные для анализа.

## Особенности

- Загрузка данных из CSV и Excel файлов
- Фильтрация по диапазону дат и городу
- Отображение метрик KPI (общая сумма продаж, средние ежедневные продажи, максимальные продажи)
- Интерактивная визуализация данных с помощью Plotly
- Поддержка мультиязычности (русский, английский, китайский)
- Возможность загрузки пользовательских данных

## Требования

- Python 3.11+
- Docker и Docker Compose (для контейнерного развертывания)

## Установка и запуск

### Локальный запуск

```bash
# Создание виртуального окружения
python -m venv venv
source venv/bin/activate  # На Linux/Mac
# или
venv\Scripts\activate  # На Windows

# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
cd web_app
streamlit run app.py
```

### Запуск с использованием Docker

```bash
# Сборка и запуск контейнера
docker-compose up --build

# Приложение будет доступно по адресу http://localhost:8501
```

## Структура проекта

- `app.py` - основной файл приложения Streamlit
- `data_loader.py` - модуль для загрузки и предварительной обработки данных
- `plotting.py` - модуль для создания визуализаций
- `analysis.py` - модуль для вычисления метрик и анализа данных
- `pages/faq.py` - страница с часто задаваемыми вопросами
- `Dockerfile` - инструкции для сборки Docker-образа
- `docker-compose.yml` - конфигурация для запуска приложения в контейнере

## Формат данных

Файл данных должен содержать следующие колонки:
- `'Дата'`: дата продажи (в формате YYYY-MM-DD)
- `'Город'`: город покупателя
- `'Имя'`: имя покупателя
- `'Фамилия'`: фамилия покупателя
- `'Сумма'`: сумма покупки (числовое значение)
- `'Валюта'`: валюта, в которой совершена покупка (например, RUB, USD, EUR)

## Агенты

Проект использует архитектуру агентов для различных задач:
- Агент анализа данных: вычисление метрик и статистический анализ
- Агент визуализации: создание графиков и диаграмм
- Агент загрузки данных: загрузка и предварительная обработка данных
- Агент интернационализации: мультиязычная поддержка интерфейса
